1 Yamaha YZF R1

Algunas características

Aerodinamismo
Yamaha YZF-R1 2015 utiliza un sensor inercial que utiliza seis ejes. Este sistema se ha desarrollado en exclusiva para esta moto y consta de un sensor giroscópico que mide la inclinación, el ángulo y la rotación además de un sensor de fuerza G que transmite datos de aceleración/frenada, izquierda/derecha y subida/bajada. Todos estos datos son analizados 125 veces por segundo. Los datos son enviados por una red CAN hasta la ECU donde se realizan los cálculos en tiempo real y se ajustan los parámetros de los diferentes sistemas electrónicos para logran un rendimiento óptimo.

Arranque
Para la salida contamos con el sistema LCS que limita las revoluciones del motor por debajo de las 10.000 (incluso accionando el acelerador al máximo). En combinación con el sistema TCS y LIF podremos concentrarnos al máximo en los segundos iniciales de una carrera. Aquí nos podremos aprovechar también del sistema de cambio rápido QSS y su sensor instalado en la palanca de cambio.

Sistema de frenado
Para parar esta bestia japonesa se recurre a un sistema ABS y al sistema unificado de frenos de Yamaha. Recordemos que este sistema, al frenad en el tren delantero también se genera fuerza de frenado en el tren trasero y cuando el piloto acciona ambos frenos el sistema se encarga de equilibrar la potencia de frenado entre ambos ejes. Este sistema utiliza el nuevo sensor IMU que tiene en cuenta diferentes datos como el ángulo de inclinación o el tipo de conducción seleccionado. Los discos delanteros son de 320 mm de diámetro y utilizan pinzas mono-bloque de anclaje radial con cuatro pistones y latiguillos de acero inoxidable. Detrás encontramos un disco de 220 mm de diámetro y una pinza con un sólo pistón.
